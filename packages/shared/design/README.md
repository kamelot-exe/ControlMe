# Design System - Minimalistic Glassmorphism

Shared design tokens and web UI primitives for the public ControlMe application.

## Structure

```text
packages/shared/design/
|-- tokens.ts
|-- tokens.json
|-- types.ts
|-- components/
|   |-- web/
|-- examples/
```

## Usage

```tsx
import { Card, Button, Input, Badge } from "@shared/design/components/web";
```

See `examples/web-example.tsx` for a complete example.
